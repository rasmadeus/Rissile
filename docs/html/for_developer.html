Описание структуры проекта.¶

Итак, папка Rissile содержит в себе:

  1. project - исходные коды проекта.

    1. .git - система контроля версий.

    2. doc - документация проекта.
      1. html - итерактивная документация проекта в формате html.
      2. source - исходный код для системы sphinx.

    3. nbproject - проектный файл для NetBeans 6.5 for Python.

    4. src - исходные коды проета.

  2. tools for project - полный и достаточный набор инструментов, необходимый для запуска проекта.
